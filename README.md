# NYC Taxi Trip Duration - Exploratory Data Analysis (EDA)

## Project Overview

This project presents an **Exploratory Data Analysis (EDA)** of the NYC Taxi Trip Duration dataset. The primary objective is to understand trip duration patterns, passenger behavior, and temporal trends using Python and data visualization techniques.

## Dataset
https://www.kaggle.com/datasets/yasserh/nyc-taxi-trip-duration

## Objectives

* Explore the distribution of trip durations.
* Analyze taxi demand by hour of the day.
* Compare trip frequency across weekdays.
* Examine monthly trip trends.
* Identify potential outliers in trip duration.
* Generate insights through visualizations.

## Visualizations

The project includes the following visualizations:

* Line Chart – Trips by Hour
* Bar Chart – Trips by Weekday
* Histogram – Distribution of Trip Duration
* Bar Chart – Monthly Trip Trend

## Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook

## Key Insights

* Most taxi trips are relatively short.
* Trip duration exhibits a right-skewed distribution with several outliers.
* Taxi demand varies throughout the day, with higher activity during peak hours.
* Weekday trip volumes differ from weekend patterns.
* Monthly trends reveal variations in taxi usage over time.

## Conclusion

This exploratory analysis provides a better understanding of taxi trip characteristics and highlights patterns that can support further predictive modeling and business decision-making.
